# Game Balance Patch Changelog

## Version 2.60 2026-04-12

### Balance Changes

## EXPERIMENTALS
**All Land / Air / Naval Experimentals**
* Removed experimental mass cost reduction
* Removed experimental energy cost reduction that was tied to reduced mass cost
* Increased smooth mass-based buildtime scaling, especially for heavy and ultra-heavy experimentals
* Reduced overall HP scaling
* HP scaling is now damped more heavily on experimentals with very high base HP
* HP scaling is now damped more heavily on experimentals with high regeneration rates

## BASE
**Cybran Shield Generators**
* `ED1` (`URB4202`): Shield `4500` -> `4000`, Recharge `15` -> `18`, Regen `75` -> `56`
* `ED2` (`URB4204`): Shield `7500` -> `7000`, Recharge `15` -> `20`, Regen `125` -> `82`
* `ED3` (`URB4205`): Recharge `16` -> `26`, Regen `156` -> `102`
* `ED4` (`URB4206`): Recharge `18` -> `28`, Regen `180` -> `132`
* `URB4207`: fixed `BlueprintId` in the QCE override from `url4207` to `urb4207`
* `URB4207`: Shield Recharge `22` -> `28`, Shield Regen `140` -> `154`

**Cybran Integrated Shield Storage**
